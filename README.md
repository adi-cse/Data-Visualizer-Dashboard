# 📊 Data Visualizer

A modern web-based **Data Visualization Tool** that allows users to upload CSV or JSON files and instantly explore their data through interactive charts and tables.
Built with vanilla JavaScript and Chart.js, this project focuses on simplicity, usability, and responsive UI design.

---

## 🚀 Features

* 📁 Upload **CSV or JSON** datasets
* 🖱 Drag & Drop file support
* 📊 Generate multiple chart types:

  * Line
  * Bar
  * Pie
  * Doughnut
  * Scatter
  * Radar
* 🔎 Preview uploaded data in table format
* 🎯 Select X-axis and Y-axis dynamically
* 🌙 Light/Dark theme toggle
* 🖼 Export chart as PNG
* 📦 Load sample dataset
* ⚡ Fast client-side processing (no backend required)

---

## 🛠 Tech Stack

* **HTML5**
* **CSS3 (Custom Theme System + Responsive UI)**
* **JavaScript (ES6 Modules)**
* **Chart.js**

## ▶️ How to Run Locally

### Option 1 — Using VS Code Live Server

1. Clone the repo
2. Open project in VS Code
3. Install **Live Server extension**
4. Right-click `index.html`
5. Click **Open with Live Server**

---

### Option 2 — Using Node Static Server

```bash
npx serve .
```

Then open the shown URL in your browser.
